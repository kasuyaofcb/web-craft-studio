# 画像ファイルの配置場所

このディレクトリには、プロジェクトで使用する画像ファイルを配置します。

## 📁 ディレクトリ構造

```
assets/images/
├── fv/              // ファーストビュー用画像
├── sections/        // セクション用画像
├── shared/          // 共通で使用する画像（ロゴ、アイコンなど）
└── icons/           // アイコンファイル
```

## 📝 命名規則

- **ファイル名**: キャメルケースまたはケバブケースを使用
  - 例: `heroImage.jpg`, `service-icon.svg`
- **拡張子**: 適切な形式を使用（`.jpg`, `.png`, `.svg`, `.webp`など）

## 🖼️ 画像の最適化

- 画像は適切なサイズにリサイズしてから配置してください
- 必要に応じて画像最適化ツールを使用してください
- WebP形式の使用を推奨（対応ブラウザの場合）

## 📌 注意事項

- 大きなファイルはGitにコミットしないでください（別途共有方法を検討）
- 著作権に注意してください
- 画像ファイル名は意味のある名前にしてください
