# 1. はじめに

> このドキュメントから順番に読んでいきましょう。

## 📋 このプロジェクトについて

このプロジェクトは、チームで1つのWebサイトを構築する実践型カリキュラムです。
模写コーディングからWordPress化まで、段階的に進めていきます。

## 🎯 学習の流れ

このプロジェクトは**4つのフェーズ**で進めます：

```
1. 環境構築・準備
   ↓
2. HTML/CSS静的コーディング（模写コーディング）
   ↓
3. WordPress化
   ↓
4. リファクタリング
```

## 🛠️ 技術スタック

- **フロントエンド**: HTML5, SCSS/CSS3, JavaScript (ES6+)
- **バックエンド**: WordPress (PHP)
- **CSS設計**: BEM + ユーティリティクラス（`u-`接頭辞）
- **CSSプリプロセッサ**: SCSS（Live Sass Compiler使用）
- **バージョン管理**: Git / GitHub
- **開発環境**: Local by Flywheel

## 📁 プロジェクト構成

### フェーズ1: 模写コーディング（静的サイト）

```
web-craft-studio/
├── index.html                     // TOPページ
├── about.html                     // Aboutページ
├── assets/
│   ├── scss/                      // SCSSファイル
│   ├── css/                       // コンパイル後のCSS（自動生成）
│   ├── js/                        // JavaScriptファイル
│   └── images/                    // 画像ファイル
└── docs/                          // ドキュメント
```

### フェーズ2: WordPress化（WordPressテーマ構造）

```
web-craft-studio/
├── style.css                      // テーマのメインスタイル
├── functions.php                  // テーマ関数
├── assets/                        // 静的アセット（フェーズ1から継続）
├── template-parts/                // セクション単位のパーツ
├── page-templates/                // 固定ページテンプレート
└── docs/                          // ドキュメント
```
